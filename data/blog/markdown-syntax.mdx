# 锚点

命名锚点使您能够跳转到同一页面上的指定锚点。例如，这些章节中的每一章：



```
# Table of Contents  * [Chapter 1](#chapter-1)  * [Chapter 2](#chapter-2)  * [Chapter 3](#chapter-3)
```


将跳转到这些部分：



```
## Chapter 1Content for chapter one.## Chapter 2Content for chapter one.## Chapter 3 Content for chapter one.
```

# 表

（建议直接用工具生成）


通过在每个单元格之间添加管道作为分隔符，并在标题下方添加一行破折号（也由条分隔）来创建表格（这行破折号是必需的）。



- 管道不需要垂直对齐。

- 表左右两侧的管道有时是可选的

- 分隔符行中的每个单元格必须使用三个或更多破折号




```
| Option | Description || ------ | ----------- || data   | path to data files to supply the data that will be passed into templates. || engine | engine to be used for processing templates. Handlebars is the default. || ext    | extension to be used for dest files. |
```

# 代码

### 内联代码
用一个反引号包裹内联代码片段：```。


例如，要`<div></div>`与其他文本内联显示，只需将其包裹在反引号中。



```
For example, to show `` inline with other text, just wrap it in backticks.
```


### 代码块
三个连续的反引号，称为“代码围栏”，用于表示多行代码：`````.


例如：



```
```html此处的示例文本...
```



```

# HTML

`<`任何介于和之间的看起来像 HTML 标记的文本`>`都将被解析为原始 HTML 标记并呈现为 HTML 而不会转义。


例子：



```
**Visit [Jon Schlinkert&#x27;s GitHub Profile](https://github.com).**
```


效果：


**Visit [Jon Schlinkert’s GitHub Profile](https://github.com).**



# 转义

任何 ASCII 标点字符都可以使用单个反斜杠进行转义。



```
\*this is not italic*
```


效果：


*this is not italic*
