<ChangelogCard type="fix" title="/search 接入 kbar 全文搜索 · Ctrl+K 修复" hash="d919495" date="2026-04-11">
命令面板里的 `/search` 一直只是跳转到 `/blog`，算是占位符。这次把它接入了 kbar，执行后会弹出全文搜索框。

<ChangelogCard type="feat" title="环形缓冲区博客 + 交互可视化" hash="a7de334" date="2026-03-25">
移植并重写了 `RingBufferViz` 和 `GoGateBufferViz` 两个交互组件。`GoGateBufferViz` 支持手动 read / write，可以一步步看位运算取模怎么工作。写这个比写文章花时间多了。

```
    ┌───┬───┬───┬───┬───┬───┐
    │ 0 │ 1 │ 2 │ 3 │   │   │
    └───┴───┴───┴───┴───┴───┘
          ▲               ▲
        read            write

  capacity = 6  (must be power of 2)
  mask     = 5  (capacity - 1)
  index    = pos & mask  ← no division
```
</ChangelogCard>

<ChangelogCard type="feat" title="全局中英文切换（i18n）" hash="6c591b5" date="2026-03-14">
没用任何外部 i18n 库，自己写了一个轻量的 `LangProvider` + `useLang()` hook，`localStorage` 持久化。约 35 个 UI 字符串都有完整中英翻译。

```typescript
const { t, lang, setLang } = useLang()

t.slashBlog   // "博客" | "blog"
t.groupPages  // "页面" | "pages"
t.noMatch     // "无匹配" | "no match"
```
</ChangelogCard>
